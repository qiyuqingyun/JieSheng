# Writer's IDE

Writer's IDE 是一个面向中文网络文学作者的本地写作工具，目标是把「章节写作」「大纲管理」「角色设定」放在同一个桌面应用里，并保持数据可控、可迁移。

应用基于 Tauri + React + TypeScript 构建，默认本地存储，不依赖云端服务。

## 项目介绍

### 适合谁
- 长篇连载作者
- 需要频繁切章节与大纲的创作者
- 希望维护角色设定一致性的团队或个人

### 解决什么问题
- 把章节、大纲、角色卡统一到一个工作流
- 减少来回切工具的中断感
- 提供自动保存与本地文件落盘，降低写作风险

### 数据存储方式
- 项目以文件夹形式存在
- 章节与大纲使用独立文件
- 角色卡使用 Markdown 文件（含结构化头信息）

## 使用教程

### 1. 启动开发环境

```bash
cd tauri-app
yarn tauri dev
```

### 2. 新建或打开项目
1. 点击顶部“新建项目”
2. 输入作品名
3. 选择保存目录
4. 或使用“打开项目”加载已有项目

### 3. 写章节
1. 左侧切到“章节”
2. 点击“+ 新建”创建章节
3. 在中间编辑区写作
4. 系统每 3 秒自动保存，窗口失焦也会自动保存

### 4. 写大纲
1. 左侧切到“大纲”
2. 点击“+ 新建”创建大纲
3. 在中间编辑区编辑大纲内容

### 5. 管理角色卡
1. 左侧切到“角色”
2. 点击“+ 新建”创建角色
3. 在角色卡中维护基础信息、状态、关系与小传
4. 点击头像可直接选择本地图片

### 6. 常用快捷键
- `Ctrl+S`：保存当前编辑内容
- `Ctrl+F`：打开搜索
- `Ctrl+Z`：撤销
- `Ctrl+Y`：重做
- `Ctrl+\` 或 `F11`：切换专注模式

### 7. 构建安装包

```bash
cd tauri-app
yarn tauri build
```

构建完成后可在打包目录找到安装程序（`.exe` / `.msi`）。

## 技术栈
- 前端：React 19 + TypeScript 5 + Tailwind CSS 4
- 编辑器：Tiptap 3
- 后端：Tauri 2 + Rust
- 构建：Vite 7
